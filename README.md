目前本项目我还在更新中，功能还在码中~

项目目前共四个分支：


### 运行项目

1. 运行后端仓库

   1.1. git clone 后端仓库地址

   1.2. cnpm install

   1.3. node app.js

2. 运行本仓库代码

   2.1. git clone 本仓库地址

   2.2. cnpm install

   2.3. npm run dev
